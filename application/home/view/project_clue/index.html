金融合伙人注册认证协议书

尊敬的FA合伙人
 
    您好

    欢迎注册成为上海歆宝信息科技有限公司（以下简称“本公司”）提供和维护的FA財平台（以下简称“本平台”）的FA合伙人。本协议由本公司为规范FA合伙人（包括个人、个体工商户和企事业单位）使用本平台的行为而制定，具有合同法律效力。
请您务必审慎阅读、充分理解各条款内容，特别是免除或者限制责任的条款、争议解决和法律适用条款。免除或者限制责任的条款可能以加粗字体显示，您应重点阅读。除非您已阅读并接受本协议所有条款，否则你无权使用本平台。您使用本平台即视为您已阅读并同意本协议的约束。

1.合作事项

  1.1 FA合伙人同意与本平台在项目合作业务方面建立合作交流关系，FA合伙人提交项目到本平台并保证所提交项目的优先性、唯一性。FA財将为FA合伙人提供FA財合伙人身份、投资机构信息、匹配合格投资方、协助尽调、制作商业计划书等服务。

2.FA合伙人的权利与义务

  2.1 积极主动配合、参与提交项目到本平台。

  2.2 可在合作范围内对外使用FA財提供的电子名片、FA合伙人身份、FA財商标、以及名称。

  2.3 不得在非合作范围外使用FA財商标、商号、名称，不得以本公司名义对外订立协议或者作出任何承诺；违反上述规定的，应由FA合伙人承担全部后果，由此造成本公司损失的，本公司有权向FA合伙人追偿。

  2.4 提交项目时应保证项目的合规合法性以及信息的真实性、完整性，不得有任何欺瞒;

  2.5 项目提交后，FA合伙人如因故需取消项目的，需提交申请经本平台审核后下架项目，否则平台有权继续上线该项目。

  2.6 如因FA合伙人履行本合同合作事项时侵害他人权益或者造成损失的，由FA合伙人自行承担，与本公司及平台无关。

3平台的权利与义务

  3.1 积极主动配合、参与本协议第一条所约定合作事项。

  3.2 平台出现匹配的投资机构时告知FA合伙人。

  3.3 如因FA合伙人履行本合同合作事项时造成本公司或平台损失的，本公司有权要求FA合伙人承担赔偿责任。

  3.4 本平台对FA合伙人提供的项目的合规合法性以及信息的真实性、完整性不承担任何保证或则责任。

  3.5本公司及平台对FA合伙人提供项目所融资的款项，不承担任何性质的担保责任。

  3.6 本公司及平台一旦发现FA合伙人存在违反本协议、本平台规定或者侵害他人权益的行为，本公司有权随时解除本协议以及与该FA合伙人的全部合作。

4.保密条款

  4.1 FA合伙人应对FA財技术资料、投资方资料等重要信息予以保密，否则，视为违约。本协议期满终止、解除或被认定无效，不影响本保密条款效力，且继续生效。

5.有效时间及续约

  5.1本协议有效期为一年，即从本协议生效之日起一年（12个月）内有效。在不满足本协议条款或法定条件情况下，任何一方在未征询另一方面同意的前提下均不得终止本协议。

6.违约责任

  6.1在本协议有效期间，双方任一方违约的，应承担违约责任，守约方有权要求违约方按有关法律规定或惯例承担违约责任。

7.不可抗力

  7.1本协议生效后，如遇火灾、水灾、战争、地震等不可抗力因素，而影响到本协议履行时，可根据情况部分或全部免予承担违约责任。

  7.2如不可抗力事故的影响造成协议无法继续正常履行，双方通过协商可终止本协议。

8.争议解决

  8.1 本协议的成立、生效、履行、解释及纠纷解决，适用中华人民共和国大陆地区法律（不包括冲突法）。

  8.2 在执行本协议中产生的争议，双方应通过友好协商解决。如果协商不能得到解决，双方可将争议提交协议签署地法院诉讼。本协议签署地为上海市浦东新区。
